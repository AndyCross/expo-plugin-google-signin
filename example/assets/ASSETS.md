# Example App Assets

This folder should contain the following placeholder images for Expo:

- `icon.png` - 1024x1024 app icon
- `splash-icon.png` - Splash screen icon
- `adaptive-icon.png` - Android adaptive icon foreground

You can create simple placeholder images or copy assets from another Expo project.

For testing, you can remove the icon references from `app.json` to skip these.
