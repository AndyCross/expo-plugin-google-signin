# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-02

### Added

- Initial release
- Expo config plugin for Android Google Sign-In via Credential Manager API
- Nonce support for secure Auth0 token exchange
- TypeScript types for credential results
- `signInWithGoogle()` - Main sign-in function
- `isGoogleSignInAvailable()` - Platform availability check
- Configurable Android package name via plugin options
- Comprehensive documentation for Auth0 integration
- Example Expo app demonstrating usage
